# Markdown синтаксис

Краткая шпаргалка по Markdown для быстрого создания документации.

## Заголовки

```markdown
# Заголовок 1 (H1)
## Заголовок 2 (H2)
### Заголовок 3 (H3)
#### Заголовок 4 (H4)
```

## Форматирование текста

```markdown
**Жирный текст**
*Курсив*
***Жирный курсив***
~~Зачеркнутый~~
`Код inline`
```

**Результат:**

**Жирный текст**  
*Курсив*  
***Жирный курсив***  
~~Зачеркнутый~~  
`Код inline`

## Списки

### Нумерованные

```markdown
1. Первый пункт
2. Второй пункт
3. Третий пункт
   1. Вложенный пункт
   2. Еще вложенный
```

### Маркированные

```markdown
- Пункт один
- Пункт два
  - Вложенный пункт
  - Еще вложенный
```

### Чекбоксы

```markdown
- [ ] Задача не выполнена
- [x] Задача выполнена
```

## Ссылки

```markdown
[Текст ссылки](https://example.com)
[Ссылка с подсказкой](https://example.com "Подсказка при наведении")
[Относительная ссылка](../other-doc.md)
[Якорь на этой странице](#заголовок)
```

## Блоки кода

### Inline код

```markdown
Используйте `команду` для выполнения.
```

### Блок кода

````markdown
```python
def hello():
    print("Hello, World!")
```
````

### Поддерживаемые языки

- `python` - Python
- `javascript` - JavaScript
- `bash` - Bash/Shell
- `yaml` - YAML
- `json` - JSON
- `markdown` - Markdown
- `html` - HTML
- `css` - CSS

## Таблицы

```markdown
| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|-------------|-------------|-------------|
| Ячейка 1    | Ячейка 2    | Ячейка 3    |
| Ячейка 4    | Ячейка 5    | Ячейка 6    |
```

**Результат:**

| Заголовок 1 | Заголовок 2 | Заголовок 3 |
|-------------|-------------|-------------|
| Ячейка 1    | Ячейка 2    | Ячейка 3    |
| Ячейка 4    | Ячейка 5    | Ячейка 6    |


## Admonitions (Material for MkDocs)

```markdown
!!! note "Заметка"
    Важная информация

!!! tip "Совет"
    Полезный совет

!!! warning "Внимание"
    Предупреждение

!!! danger "Опасно"
    Критическое предупреждение

!!! info "Информация"
    Дополнительная информация
```

!!! note "Заметка"
    Важная информация

!!! tip "Совет"
    Полезный совет

## Сноски

```markdown
Текст со сноской[^1]

[^1]: Текст сноски
```

## Аббревиатуры

```markdown
*[HTML]: HyperText Markup Language
*[CSS]: Cascading Style Sheets

Первое упоминание HTML и CSS будет с подсказкой.
```

## Комментарии

```markdown
<!-- Это комментарий, не будет виден в HTML -->
```

## Escape-символы

Чтобы показать специальные символы как есть, используйте `\`:

```markdown
\* Не будет списком
\# Не будет заголовком
```
